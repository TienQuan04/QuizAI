TROUBLESHOOTING
===============

Problem: Gradient not showing
→ Check gradient_header.xml exists in drawable folder
→ Rebuild project
→ Run on device (API 16+)

Problem: Avatar not showing
→ Check ic_avatar_placeholder.xml exists
→ Verify ImageView background and src are set
→ Rebuild project

Problem: Statistics not loading
→ Check userId is valid (not -1)
→ Verify database has quiz history data
→ Check DatabaseHelper methods work
→ Test database queries

Problem: Progress bar not showing
→ Check progress_bar_bg.xml exists
→ Verify setProgress() called in code
→ Check max="100" in ProgressBar XML

Problem: Buttons not working
→ Verify findViewById() IDs match XML
→ Check onClick listeners attached
→ Test on actual device (emulator can be slow)

Problem: Compile errors
→ Check AndroidX CardView dependency in build.gradle
→ Rebuild project
→ Check all imports are correct


CUSTOMIZATION QUICK TIPS
========================

Change Header Colors:
Edit: app/src/main/res/drawable/gradient_header.xml
Change: android:startColor="#2196F3"
To: your_desired_color

Change Button Colors:
Edit: app/src/main/res/layout/activity_profile.xml
Find: android:backgroundTint="#2196F3"
Change: to your_color

Change Icon Symbols:
Edit: app/src/main/res/layout/activity_profile.xml
Find: android:text="📋" (or other emoji)
Change: to different emoji

Change Statistics Labels:
Edit: app/src/main/res/layout/activity_profile.xml
Find: android:text="Total Quizzes"
Change: to your_label
