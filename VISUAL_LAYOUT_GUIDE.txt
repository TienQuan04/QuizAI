CUSTOMIZATION GUIDE
===================

To Change Colors:
1. Update gradient_header.xml for header colors
2. Modify stat_icon_bg_*.xml for icon backgrounds
3. Edit activity_profile.xml hex color values
4. Update button colors in buttons section

To Change Dimensions:
1. Header height: Change 280dp in LinearLayout
2. Card corners: Modify cardCornerRadius value
3. Icon size: Change 60x60dp dimensions
4. Padding: Adjust padding values in containers

To Add New Cards:
1. Copy CardView block in activity_profile.xml
2. Create new icon background drawable
3. Update ProfileActivity.java to populate values
4. Add corresponding database queries if needed
