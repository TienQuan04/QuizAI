DEBUGGING GUIDE
===============

If Gradient Not Showing:
1. Check gradient_header.xml file exists
2. Verify @drawable/gradient_header reference
3. Check Android version (should work on 5.0+)
4. Rebuild project

If Avatar Not Showing:
1. Check ic_avatar_placeholder.xml exists
2. Verify @drawable/ic_avatar_placeholder reference
3. Check ImageView background and src are set
4. Verify file is in drawable folder

If Cards Not Showing Shadows:
1. Check CardView version in build.gradle
2. Verify elevation="6dp" in XML
3. Enable hardware acceleration (usually on by default)
4. Check min API level supports CardView (21+)

If Progress Bar Not Showing:
1. Check progress_bar_bg.xml exists
2. Verify android:progressDrawable reference
3. Set progress value in code (accuracyProgressBar.setProgress())
4. Check max="100" in ProgressBar widget

If Buttons Not Responding:
1. Check View IDs match in XML and Java
2. Verify findViewById() calls correct
3. Check onClick listeners attached
4. Test on device (emulator sometimes has lag)

If Numbers Not Displaying:
1. Check DatabaseHelper methods return correct values
2. Verify userId is valid (-1 check)
3. Check setText() is called with String values
4. Verify database has data

If Logout Not Working:
1. Check SharedPreferences editor.apply() is called
2. Verify Intent flags are correct
3. Check LoginActivity exists
4. Test on device, not emulator

   Change: android:startColor="#2196F3"
   To: desired_color
   Change: android:endColor="#9C27B0"
   To: desired_color

2. Change Button Colors:
   Edit: activity_profile.xml
   Change: android:backgroundTint="#2196F3"
   To: desired_color

3. Change Corner Radius:
   Edit: activity_profile.xml
